# AUTO_FRONT_SCREENPLAY

Proyecto de automatización UI centrado en el usuario utilizando el patrón **Screenplay**.

## 🛠️ Tecnologías
- **Java 17**
- **Serenity BDD (Screenplay)**
- **Gradle**
- **Cucumber**

## 🎯 Objetivo
Demostrar escalabilidad y legibilidad mediante el uso de **Actores, Tareas y Preguntas**.

## 🚀 Ejecución
```bash
./gradlew clean test aggregate
```
Reportes en `target/site/serenity/index.html`.
